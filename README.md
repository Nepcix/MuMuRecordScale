# MuMuRecordScale

Keywords : MuMu , Record , Resolution , Scale , Zoom , WPF

MuMu模拟器操作录制缩放工具

适用于在MuMu模拟器中录制操作后模拟器分辨率修改了导致操作录制无法适配的问题。

## 下载 ― Download

[Download](https://github.com/Nepcix/MuMuRecordScale/releases/download/v1.0.0/MuMuRecordScale.zip)

解压即用

如不能运行请自行安装 .Net Framework 4.7.2

## 用法 ― Usage

1.打开操作录制文件

2.输入原始分辨率和调整后分辨率

3.调整并保存文件
